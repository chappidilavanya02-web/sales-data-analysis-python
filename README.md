# Sales Data Analysis using Python

## Project Overview
This project performs exploratory data analysis (EDA) on a sales dataset to understand
sales trends, customer behavior, and product performance using Python.

The analysis focuses on year-wise sales, brand and category performance,
customer ratings, and demographic insights through clear visualizations.

---

## Dataset
The dataset contains the following key attributes:
- item_id
- user_id
- rating
- timestamp
- gender
- category
- brand
- year
- month
- quantity
- unitprice
- amount

---
